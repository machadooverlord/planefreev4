# Plane Free v4

Shoot 'em up espacial roguelike com sistema de cartas e evolução de naves.

## 🎮 Como Rodar
```bash
# Clonar repositório
git clone https://github.com/machadooverlord/planefreev4.git
cd planefreev4

# Instalar dependências
pip install -r requirements.txt

# Rodar jogo
python src/main.py
```
